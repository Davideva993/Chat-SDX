    Key Principles

    • Ask users only for data strictly required for functionality
    • Connections run through the Tor network
    • All messages are end-to-end encrypted; the server never sees the keys
    • The server is assumed to be potentially compromised
    • All client-side data is stored ephemerally — cleared on page reload
    • A "Destroy Chat" button instantly wipes the session and reloads the page

    Host a chat: next steps:

    1. Choose the room code and the secret words
    2. Click "host this chat"
    3. Share the room code and secret words with the other user via a temporary secure channel
    4. Wait for the other user to join

    If the page reloads, the room is lost.

    Join a chat: next steps:

    1. Use a temporary secure channel to obtain the room code and secret words
    2. Enter the information and click “Join this chat”

    If the page reloads, the room is lost.
